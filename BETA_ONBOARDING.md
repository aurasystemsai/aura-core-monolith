## üõ†Ô∏è Quick Start Guide

### Connect Your Store (Shopify)

**Option 1: Shopify App (Recommended)**
```
1. Go to Dashboard ‚Üí Settings ‚Üí Integrations
2. Click "Connect Shopify Store"
3. Enter store URL: yourstore.myshopify.com
4. Authorize app
5. Done! Data syncs automatically
```

**Option 2: API Key**
```
1. Shopify Admin ‚Üí Apps ‚Üí Develop apps
2. Create custom app: "AURA Platform"
3. Request scopes: read_products, read_orders, read_customers
4. Copy API key & password
5. Paste into AURA ‚Üí Settings ‚Üí API Keys
```

**Option 3: Demo Data**
```
Don't have a store? No problem!
Click "Use Demo Data" in onboarding
Get fully populated sample store to explore
```

### First Automation: Dynamic Pricing

**5-Minute Setup:**
```
1. Tools ‚Üí Dynamic Pricing Engine
2. Click "Create Rule"
3. Name: "Competitor Price Match"
4. Condition: If competitor price < our price
5. Action: Match competitor price - $0.50
6. Click "Activate"
7. Done! Prices auto-update every hour
```

### First Campaign: Abandoned Cart Email

**10-Minute Setup:**
```
1. Tools ‚Üí Email Campaign Builder
2. Templates ‚Üí "Abandoned Cart Series"
3. Customize emails (3 emails over 3 days)
4. Set discount: 10% off first reminder
5. Click "Activate Flow"
6. Done! Campaigns send automatically
```

### First Content Brief: Blog Post

**3-Minute Setup:**
```
1. Tools ‚Üí AI Content Brief Generator
2. Enter keyword: "best running shoes 2026"
3. Click "Generate Brief"
4. Get: Title suggestions, outline, keywords, SERP analysis
5. Export to Google Docs or copy
6. Done! Write blog post from brief
```
